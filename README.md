# Analyzing and Forecasting Glacier Mass Loss in the Himalayas Using Machine Learning Models

## Overview

This repository contains the dissertation, source code, visualisations, and supporting materials for my MSc Data Analytics project.

The study investigates glacier mass loss across High Mountain Asia using data analytics, interactive visualisation, time-series analysis, and machine learning models. Glacier inventory data and historical glacier mass-balance observations were analysed to understand regional glacier behaviour and forecast future glacier mass loss.

## Research Objectives

- Analyse glacier characteristics across High Mountain Asia
- Compare glacier distribution between Central Asia, South Asia West, and South Asia East
- Study glacier mass-balance trends from 1990–2025
- Build forecasting models for glacier mass loss
- Compare machine learning model performance
- Identify regions at higher risk of glacier loss

## Machine Learning Models

The following models were evaluated:

- Linear Regression
- Random Forest Regression

Random Forest achieved better predictive performance than Linear Regression based on:

- Mean Absolute Error (MAE)
- Root Mean Squared Error (RMSE)
- R² Score

## Key Findings

- Glacier mass loss varies significantly across High Mountain Asia.
- Central Asia contains the largest glacier inventory.
- Western and Eastern Himalaya exhibit the greatest glacier mass loss.
- Glacier mass balance has become increasingly negative in recent years.
- Random Forest Regression outperformed Linear Regression for glacier mass-balance prediction.
